Bobby's diner

Décembre 2023

HTML CSS Javascript Photoshop Illustrator

Ce projet avait pour but de créer un site web pour un foodtruck fictif de notre choix. Pour ma part, j'ai décidé de créer, Bobby's Diner, un foodtruck spécialisé dans les burgers et les milkshakes. J'ai réalisé ce projet en utilisant HTML, CSS et Javascript pour le développement du site web et Photoshop et Illustrator pour la création des visuels. J'ai d'abord établi un moodboard pour pouvoir définir l'identité visuelle du foodtruck. J'ai ensuite réalisé les maquettes du site sur Figma avant de commencer à le développer pour de vrai. Ce site m'a permis de découvir le format JSON qui m'a permis de concevoir un menu qui se change de manière dynamique en fonction des données du fichier JSON. Ce projet m'a également permis de redécouvrir la librairie Javascript Leaglet qui m'a permis d'intégrer une carte interactive pour localiser le foodtruck.
